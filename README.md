🎬 VideoQuiz AI
Automatically summarize any video and generate interactive quizzes from its content


📌 Overview
VideoQuiz AI is a web application that takes a video URL, generates a concise AI-powered summary of its content, and automatically creates a multiple-choice quiz to test comprehension. Built with Python and Flask.

Portfolio project demonstrating NLP, API integration, and full-stack Python web development.


✨ Features

🎥 Video Summarization — Paste a YouTube URL and get an AI-generated summary
🧠 Quiz Generation — Auto-generates MCQs based on the video content
🔐 Secure API Handling — API keys managed via environment variables
🌐 Deployable — Ready to deploy on Render, Railway, or any cloud platform
